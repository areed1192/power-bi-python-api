# Installation

## From PyPI

```console
pip install python-power-bi
```

## From Source (editable)

```console
git clone https://github.com/areed1192/power-bi-python-api.git
cd power-bi-python-api
pip install -e .
```

## Development Install

To install with development tools (linting, testing, type checking):

```console
pip install -e ".[dev]"
```

## Requirements

- Python >= 3.10
- `msal >= 1.35.1`
- `requests >= 2.33.0`
